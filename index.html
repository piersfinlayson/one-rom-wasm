One ROM WASM Library

Main Website
About

This site hosts One ROM Web Assembly library, which can be used to:

  • Retrieve properties of generic ROMs, including 23 mask programmed and 27 EPROMs
  • Parse One ROM firmware .bin files
  • Parse One ROM firmware read from flash
  • Retrieve information about One ROM hardware versions and supported MCUs
  • Generate One ROM firmware metadata and ROM image files (coming soon)
