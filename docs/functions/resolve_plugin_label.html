    • resolve_plugin_label(
          slot_index: number,
          source: string,
          fetch_callback: Function,
      ): Promise<any>

      Resolve a device plugin slot's image source to display information.

      slot_index is the plugin's slot (0 = system, 1 = user); since a device's plugins are reported in slot order, the caller can pass the plugin's index within the plugins list. source is the image source the device recorded. fetch_callback is a JS async function (url: string) => Promise<Uint8Array>, used only for official plugins, to fetch the release manifest for the display name and description.

      The manifest fetch is best-effort: on any failure the label falls back to the slug, so this never rejects on a network error. Returns JS null only when slot_index is not a plugin slot.
